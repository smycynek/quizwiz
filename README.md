# Quizwiz

# A front-end for QuizFreak, a quiz-building app

I wanted to create a full stack project from scratcththat included

* UI (React)
* State management (Redux and ReduxSauce)
* API Workflow (Sagas)
* Back-End (Django)
* Deployment (Heroku)
